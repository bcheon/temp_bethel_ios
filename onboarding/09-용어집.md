# 09. 용어집

## 푸쉬 알림 관련

| 용어 | 풀네임 | 한 줄 설명 |
|------|-------|----------|
| **APNs** | Apple Push Notification service | Apple 의 iOS 푸쉬 게이트웨이. 모든 iOS 푸쉬는 여기를 거침. |
| **FCM** | Firebase Cloud Messaging | Google 푸쉬 SDK. iOS에선 APNs 위에 추상화 레이어 + 토픽 구독 + HTTP API 제공. |
| **FCM HTTP v1** | — | FCM 의 신 API. JWT/OAuth2 인증. legacy server key 대체 (2024+ 권장). |
| **APNs token** | Device Token | Apple 이 디바이스에 발급하는 64-byte hex. iOS가 `didRegisterForRemoteNotifications`로 전달. |
| **FCM token** | Registration Token | FCM 이 발급하는 ~150자 토큰. APNs token + Firebase Sender ID 매핑. 서버는 이걸 보관. |
| **Topic** | FCM Topic | FCM 의 pub/sub 채널. 디바이스가 구독 → 토픽 publish 시 일괄 수신. 그룹 = 토픽 1:1 매핑. |
| **Condition** | FCM Condition String | 2~5개 토픽을 OR/AND 조합 (예: `'bkc_youth' in topics \|\| 'bkc_newfam' in topics`). |
| **mutable-content** | APS payload key | 1로 설정 시 NSE 가 wake 됨 (콘텐츠 수정 + 텔레메트리 가능). |
| **Notification (alert)** | — | 배너/잠금화면에 표시되는 알림. APS의 `alert` 키. |
| **Data message** | — | 시스템 알림 표시 안 하고 앱에 데이터만 전달. v1에서 안 씀 (mixed로 보냄). |
| **NSE** | (UN)Notification Service Extension | iOS 앱 부속 프로세스. 푸쉬 도착 시 30초 budget으로 콘텐츠 변형 + 네트워크 호출 가능. |

## iOS / Apple 생태계

| 용어 | 풀네임 | 한 줄 설명 |
|------|-------|----------|
| **AASA** | Apple App Site Association | `https://<domain>/.well-known/apple-app-site-association` JSON. Universal Link 매핑 정의. |
| **Universal Links** | — | URL 스킴(myapp://) 대신 `https://` URL 이 앱을 직접 열어주는 Apple 메커니즘. SEO + 폴백 친화적. |
| **Bundle Identifier** | — | 앱 unique ID. 예: `org.bkc.churchapp`. Apple 전체에서 unique 해야 함. |
| **Team ID** | Apple Developer Team ID | 10자 영숫자. Apple 계정 단위. `YOUR_TEAM_ID` placeholder 자리. |
| **Provisioning Profile** | — | 어떤 Team ID + Bundle ID + 인증서 + 디바이스가 빌드 가능한지 정의. |
| **App Group** | — | NSE 와 메인 앱이 데이터 공유하는 entitlement. v1.0 에선 `group.org.bkc.churchapp`. |
| **Keychain** | — | iOS 의 보안 저장소. 앱 재설치 시에도 보존 가능 (정책 따라). 디바이스 UUID 저장 위치. |
| **Entitlements** | `.entitlements` 파일 | 앱이 어떤 Apple 시스템 권한 (Push, App Group, Sign in with Apple) 사용하는지 선언. |
| **TestFlight** | — | App Store 출시 전 베타 배포 채널. 내부 100명 + 외부 10000명. |
| **App Store Connect** | — | Apple 의 개발자 백오피스. 앱 메타데이터 + 빌드 업로드 + 심사. |
| **APS** | Apple Push Service payload | 푸쉬 페이로드 안의 `aps` 딕셔너리. alert, sound, badge, mutable-content 등. |
| **fastlane** | — | iOS/Android 자동 배포 Ruby 툴. `match` (인증서), `gym` (빌드), `pilot` (TestFlight). |
| **XcodeGen** | — | YAML (`project.yml`) → `.xcodeproj` 변환기. xcodeproj git diff 지옥 회피. |
| **SPM** | Swift Package Manager | Xcode 내장 의존성 관리. CocoaPods 의 후속. Firebase 12.x 가 SPM 지원. |
| **XCTest** | — | Apple 표준 유닛 테스트 프레임워크. iOS 의 JUnit/PHPUnit 등가물. |
| **XCUITest** | — | XCTest 위 UI 자동화. Espresso/Selenium 등가물. |

## WordPress / PHP

| 용어 | 한 줄 설명 |
|------|----------|
| **WP-cron** | WP 가짜 cron — HTTP 트래픽 들어올 때만 실행. 시스템 cron으로 백업 권장. |
| **Action Scheduler** | WooCommerce 발 lib. WP-cron 보다 안정적인 백그라운드 작업 큐. 이 프로젝트가 사용. |
| **dbDelta** | WP 의 스키마 마이그레이션 헬퍼. 안전한 ALTER (컬럼 추가) 만 하고 drop 안 함. |
| **`$wpdb`** | WP 글로벌 DB 객체. PDO 비슷한 wrapper + WP 보안 (prepare). |
| **`prepare()`** | `$wpdb->prepare()`. SQL placeholder 바인딩. **필수** — 직접 보간하면 SQL injection. |
| **Nonce** | "Number used once" — WP의 CSRF 토큰. 폼/AJAX 보안 검증. |
| **Capability** | WP 권한 시스템. `manage_options` = 관리자 권한. |
| **REST API namespace** | `wp-json/bkc/v1` 의 `bkc/v1` 부분. 플러그인 격리. |
| **Transient** | WP 단기 캐시. 키 + TTL. memcached/Redis 사용 가능 (없으면 wp_options 사용). |
| **PHPCS** | PHP_CodeSniffer. WP Coding Standards 검사. |
| **Composer** | PHP 패키지 매니저. npm 의 PHP 버전. |
| **Service Account** | Google Cloud 의 머신용 계정. JSON 키 발급 → JWT 서명에 사용. |

## 이 프로젝트 고유 용어

| 용어 | 의미 |
|------|------|
| **BKC** | Bethel Korean Church (Irvine). 베델교회 어바인. |
| **Group** | 사용자가 구독하는 카테고리. 현재 3개: `all`, `youth`, `newfamily`. |
| **Topic** | FCM 토픽. 그룹과 1:1. `youth` → `bkc_youth`. |
| **Campaign** | 한 번의 발송 단위. UUID 로 식별. `wp_bkc_campaigns` 의 한 row. |
| **Subscriber** | 푸쉬 등록한 디바이스 1대. `wp_bkc_subscriptions` 의 한 row. device_id 가 PK. |
| **Event** | 디바이스에서 발생한 push 라이프사이클 이벤트. delivered/opened/deeplinked. |
| **IRON RULE** | 회귀 절대 금지인 7개 테스트. 하나라도 깨지면 PR 머지 차단. |
| **Idempotency Token** | 폼 hidden input. 같은 토큰으로 두 번 submit 해도 캠페인 1개. |
| **Dispatcher** | 캠페인 → FCM 발송 오케스트레이션 클래스. Action Scheduler hook. |
| **Stats Rollup** | 매시간 raw event → 집계 카운터 변환. 멱등 (re-run 안전). |
| **Onboarding** | 앱 첫 실행 시 그룹 선택 화면. 완료해야 푸쉬 받음. |
