## 한 페이지 요약

- **목표:** 카카오톡 채널 의존도를 끊고, 자체 푸쉬 인프라(iOS 앱 + WordPress 발송 콘솔)로 교회 공지를 송출.
- **사용자:** iOS 16+ 사용 교인 (Android는 v1.1 로드맵).
- **발송자:** WP 관리자 권한을 가진 교회 IT팀 / 사역자.
- **인프라:** 교회가 이미 운영하는 `bkc.org` WordPress + Firebase(FCM) 무료 등급.
- **현재 상태:** v1.0.0 구현 완료. 78개 자동화 테스트 통과. App Store 제출 직전.
