# temp_bethel_ios

An iOS project repository.

## Requirements

- Xcode 15.0+
- iOS 17.0+
- Swift 5.9+

## Getting Started

1. Clone the repository
   ```sh
   git clone https://github.com/bcheon/temp_bethel_ios.git
   ```
2. Open the `.xcodeproj` or `.xcworkspace` file in Xcode
3. Build and run on a simulator or device
